# **Gremlin – Chaotic AI Companion**

## Overview

**Gremlin** is a sarcastic, chaotic-energy AI chatbot for Android.
It blends humor, mischief, and personalized sarcasm into a mobile app with:

* Adaptive chaos-based personality
* Context-aware catchphrases
* Smart notifications & scheduled reminders
* Floating “Summon the Gremlin” widget
* Admin dashboard with performance metrics

Built via **Google Antigravity**, Gremlin outputs a fully functional **APK** while allowing backend tweaks and personality customization.

---

## Features

* **Chaotic Gremlin Chatbot** – mischievous, sarcastic, and responsive
* **Admin Dashboard** – tracks engagement, chaos score, sentiment, and metrics
* **Smart Notifications** – scheduled and adaptive to user behavior
* **Summon the Gremlin Widget** – quick access to chaos at any time
* **Local & Cloud Hybrid** – runs offline or uses backend LLM if needed
* **Feedback System** – lets users rate responses and tune the chaos

## Installation & Usage

1. Clone the repo:

```bash
git clone https://github.com/username/gremlin.git
```

2. Open in **Android Studio** or **VS Code**
3. Build the APK or run on an emulator/device
4. Admin dashboard: triple-tap app title to unlock
5. “Summon the Gremlin” widget: drag and tap for quick interactions
